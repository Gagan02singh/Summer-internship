Attachments in the package
1. python code - Screenshot_optimized_v2.py
2. csv file - waybackLinkV1.csv

Content you need to install externally 
phantom driver - driver url - for mac - https://bitbucket.org/ariya/phantomjs/downloads/phantomjs-2.1.1-macosx.zip
for windows - https://bitbucket.org/ariya/phantomjs/downloads/phantomjs-2.1.1-windows.zip

To succesfully run the code, you need to change values of 3 variable given below with correct path:-

path_to_screenshot = path where you want screenshots to be saved
eg: path_to_screenshot = 'D:/screenshots'

path_to_csv = path where csv file is stored
eg: path_to_csv = 'D:/Summer_RA/Code/waybackLinkV1.csv'

path_to_driver = path to .exe file of the driver
eg: path_to_driver = 'C:/Users/singh/Downloads/phantomjs-2.1.1-windows/phantomjs-2.1.1-windows/bin/phantomjs.exe'
